# VSCode Shortcuts

| **Shortcut**   | **Description**   |
| --------------|-------------------|
| `ctrl+a` `ctrl+k` `ctrl+0` | Fold All Functions |
| `alt+shift+F12,` | Find All References |
| `ctrl+/,` | Comment out |
| ```ctrl+` ``` | Open Terminal |
| `ctrl+shift+\` | Find Closing Bracket |
